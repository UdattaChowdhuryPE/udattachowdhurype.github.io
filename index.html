About

I'm an AI Research Engineer at Atomicwork, where I built the multi-agent runtime powering our agentic service management product — a Gateway-based harness with per-agent context isolation, MCP server integrations, YAML workflow orchestration, and subagent spawning.

The core problems I live in: stateful multi-agent coordination, domain-specific agent evaluation, and code agent harness design. Before Atomicwork, I was at SuperAGI Research building autonomous coding agents (SuperCoder 2.0 — #1 open-source on SWE-Bench), cognitive GUI automation (AUTONODE), and multi-agent orchestration frameworks.

I studied ECE at Netaji Subhas University of Technology (Delhi), graduating with dual minors in ML and Signal Processing, ranked 3rd in my department. I've also done research stints at ISRO and NTNU Norway.

Outside of work, I write — poetry, prose, and the occasional piece on things that aren't code.
